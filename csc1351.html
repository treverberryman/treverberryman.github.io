CSC 1350 & 1351

CSC 1350 & 1351 – Computer Science I & II for Majors

Course Description - Introduction to Java. Practice simple Java Programs.

Assignments – Merge-Sort, Binary Search, Simple Calculator, Gregorian Date Calendar, and more...

Course Reflection – In these courses, I learned how to program several basic Java algorithms. Develop solutions to problems using an object-oriented approach and emphasizes the concepts of recursion; dynamic memory; data structures (lists, stacks, queues, trees); exception handling.
