CSC4356

Interactive Computer Graphics

Course Description: This course provides and introduction to 3D computer graphics, emphasizing real-time graphics programming using WebGL and JavaScript programming language. Topics include an introduction to JavaScript, the OpenGL ES shading language, the mathematics of 3D graphics, transformation, texturing, lighting,
scene mgmt., and interaction. Topics in software engineering, including project mgmt. and source code control, are also covered. Students may expect to gain experience developing interactive 3D content running across Windows, Linux, or OS X in a web browser on their PC, laptop, or mobile device. No prior experience with 3D graphics is required.
